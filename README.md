# vue-custom-mobilenet-prediction

Vue.js app with TFjs performing real-time inference on webcam feed with a custom mobilenet

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
